Di awal kita harus "git init" dulu

di lanjut dengan "git add test.html" atau untuk melakukan ke semua file "git add ."

untuk melihat info status nya kita bisa menggunakan"git status"

di lanjut dengan "git commit -m "test"

untuk melihat info commit, kita bisa menggunakan"git log"

ini untuk mengetes commit ke 2 kali

ulangi dari git add jika ingin commit lagi

ini akan kita coba revert yang beneran

kita akan masuk ke materi branch

untuk mengecek nama nama branch kita, kita bisa menggunakan "git branch"

untuk membuat branch baru, kita bisa menggunakan "git branch branch-name"

untuk membuat branch baru namun langsung pindah ke branch tersebut kita bisa menggunakan "git branch -b branch-name-new"

untuk masuk ke branch kita, kita bisa menggunakan "git checkout branch-name"

untuk menghapus branch kita, kita bisa menggunakan "git branch -d branch-name"

untuk memasukkan kode ini ke dalam branch master, kita bisa menggunakan "git merge branch-name"

namun kita pindah dulu ke branch master untuk melakukannya!

jika terjadi konflik saat melakukan merge, kamu bisa memilih manual di kode nya yang mana yang kamu inginkan

Belajar di bagian web github, repo

kita membuat dulu repo baru pada web github

lalu kita konek di bagian terminal dengan mengetik "git remote add [nama yang kita ingikan] linkReponya"

lalu cara kita untuk menghubungkan file file disini ke repo dengan berikut :

git remote set-url [nama remotenya] linkReponya namun ada yang dirubah seperti contoh dibawah ini

https://MDhafaR@github.com/MDhafaR/pokoknya-belajar-github.git

pada kode di atas anda bisa melihat bahwa sebelum github.com kita harus memasukkan username github kita dilanjut dengan @

lalu kita baru bisa push dengan menggunakan "git push [nama remote] [nama branch]" contoh "git push belajarGitHub master"

untuk memudahkan push kita bisa ketik "git push" saja setelah melalui step step sebelumnya

jika kita ingin memasukkan branch baru ke dalam gituhub kita, kita bisa menggunakan "git push [remote] [branch-name]"
