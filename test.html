Di awal kita harus "git init" dulu

di lanjut dengan "git add test.html" atau untuk melakukan ke semua file "git add ."

untuk melihat info status nya kita bisa menggunakan"git status"

di lanjut dengan "git commit -m "test"

untuk melihat info commit, kita bisa menggunakan"git log"

ini untuk mengetes commit ke 2 kali

ulangi dari git add jika ingin commit lagi

ini akan kita coba revert yang beneran

kita akan masuk ke materi branch

untuk mengecek nama nama branch kita, kita bisa menggunakan "git branch"

untuk membuat branch baru, kita bisa menggunakan "git branch branch-name"

untuk membuat branch baru namun langsung pindah ke branch tersebut kita bisa menggunakan "git branch -b branch-name-new"

untuk masuk ke branch kita, kita bisa menggunakan "git checkout branch-name"

untuk menghapus branch kita, kita bisa menggunakan "git branch -d branch-name"

untuk memasukkan kode ini ke dalam branch master, kita bisa menggunakan "git merge branch-name"
